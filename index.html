Library Reference

Every tool in NVIDIA's robotics stack — explained for sidewalk delivery

Jetson Orin NX
Hardware
The brain that doesn't blink under pressure

A compact system-on-module delivering 100 TOPS of AI performance in a form factor small enough for a 50-lb delivery robot. It combines a GPU, CPU, DLA (Deep Learning Accelerator), and dedicated vision engines on one chip, enabling multiple neural networks to run simultaneously without thermal or power collapse. Unlike Hailo-8's fixed-function accelerator, Orin's heterogeneous architecture lets developers retarget compute dynamically between perception, planning, and control tasks.

Sidewalk Delivery Use Case
Runs 3D object detection, pedestrian intent prediction, and path planning in parallel on a Starship-sized robot — tasks that would serialize and stall on a Hailo-8 during dense pedestrian crossings.
Isaac Sim
Simulation
A photorealistic virtual world for training and testing

Isaac Sim is NVIDIA's robotics simulation platform built on Omniverse. It generates physically accurate, ray-traced environments where robots can be trained and validated before a single real-world test. Sensor models inside Isaac Sim replicate real LiDAR, camera, and IMU noise profiles, so models trained in simulation transfer reliably to hardware. Thousands of edge cases — slippery sidewalks, crowds, cyclists — can be run overnight in parallel.

Sidewalk Delivery Use Case
A delivery company can simulate a new city's street layout, pedestrian density, and weather patterns entirely in Isaac Sim before dispatching a single robot — compressing a 6-month city onboarding into days.
Isaac Lab
Simulation
Where robots learn by doing — at scale

Isaac Lab is NVIDIA's reinforcement learning framework built on top of Isaac Sim. It lets engineers define reward functions for robot behaviors and train them across thousands of parallel simulation instances simultaneously. This GPU-accelerated RL pipeline dramatically compresses training time: behaviors that would take weeks on CPU clusters train in hours. Isaac Lab integrates directly with Isaac Sim's physics engine for accurate contact dynamics.

Sidewalk Delivery Use Case
Training a robot to cross a busy intersection safely requires millions of attempts. Isaac Lab runs 10,000 parallel intersection scenarios on one GPU, teaching a delivery bot recovery behaviors that Starship engineers currently have to hard-code manually.
DriveWorks SDK
Software
Sensor fusion infrastructure that just works

DriveWorks is NVIDIA's middleware for autonomous machine perception — it provides optimized pipelines for camera, radar, LiDAR, and ultrasonic sensor fusion, calibration, and synchronization. Engineers don't re-implement low-level sensor drivers or time-sync logic; DriveWorks handles it with hardware-accelerated primitives tuned for Jetson SoCs. It also provides surround-view stitching, freespace detection, and object tracking out of the box.

Sidewalk Delivery Use Case
A Starship robot using DriveWorks can fuse data from 9 cameras and ultrasonic sensors with sub-millisecond synchronization, enabling it to detect and track a running child from multiple angles simultaneously — something stereo-only setups miss.
TensorRT
Software
Neural network speed, extracted to the last nanosecond

TensorRT is NVIDIA's inference optimization library that takes a trained neural network and compiles it into an engine maximally tuned for the target GPU/DLA hardware. It performs layer fusion, precision calibration (FP16/INT8), and kernel auto-tuning, often doubling or tripling throughput compared to running the same model in PyTorch. On Jetson Orin NX, TensorRT engines can be split across GPU and DLA cores to maximize parallel throughput while minimizing power.

Sidewalk Delivery Use Case
The pedestrian detection model that takes 18ms in PyTorch on Orin runs in 4ms as a TensorRT INT8 engine — giving a delivery robot the reaction speed it needs to brake safely at a 5 mph rolling approach to a crosswalk.
NVIDIA Cosmos
Simulation
A world model that understands physics before robots do

Cosmos is NVIDIA's foundation world model for physical AI — a large generative model trained on massive amounts of real-world video data that understands how the physical world behaves. It can generate realistic synthetic training data, predict plausible future states of a scene, and serve as a pre-trained backbone for downstream robotics perception tasks. Unlike domain-specific simulators, Cosmos understands open-world dynamics including weather, crowds, construction zones, and novel object types.

Sidewalk Delivery Use Case
When a delivery fleet encounters a never-before-seen obstacle type — a food truck parked mid-sidewalk, scaffolding, a street fair — Cosmos can generate thousands of training examples of that scenario without a human driving robot into real copies of it.
